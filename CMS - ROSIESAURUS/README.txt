This is a version of a CMS created for sample purposes. 

Users can login, add data to their personal profile, and delete information. Users can recieve notifications and store information. 

This application demonstrates usage of:

1. PHP login functionality
2. Form validation
3. Connection to mySQL database
4. PHP security (SQL injections & password encryption)
5. Comment system creation
6. Sending and reciving emails from within the CMS using PHP & Javascript
7. Real time notifications
8. Multi-language functionality
9. Dynamic pages - Separate admin for logged in users
10. Used: PHP, JS, SQL, JQUERY, AJAX, Bootstrap
